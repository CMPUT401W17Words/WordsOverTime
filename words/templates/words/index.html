What is WordsOverTime?

WordsOverTime is a web based database application that involves running a series of requests over a database that stores several articles, and outputting the results of those requests to the user.

Designed for Reseachers

Designed for researchers who are interested in analyzing word meanings over time.

Time Saver

An efficient way to acquire the desired values from a large database.

UX-centred

User experience centred UI design and color schema.

Mobile-friendly

A responsive web framework allows this page to fit many different devices.

Easy to use

Within only a few clicks, queries are easy for even students to use.

Benefit

Benefiting for understanding how words are used by media over time

Analyze





N Closest Neighbours

The N closest neighbours in each output m atrix for a given list of words.

Format as whitespace seperated words. Choose either a file upload or use the text box.




Cosine Distance of Word Pairs

The cosine distance in each output matrix for a given list of word pairs. Format as pairs of words joined by a , with pairs seperated by spaces. Choose either a file upload or use the text box.


Average tfidf

A measure of the average tdidf over time.

Format as whitespace seperated words.

Pairwise Conditional Probabilities

Pairwise conditional probablities by article for each corpus chunk.

Format as pairs of words joined by a , with pairs seperated by spaces. Choose either a file upload or use the text box.

Valence and Arousal

The Average valence and arousal of documents, optionally only those containing certain words.

Format as whitespace seperated words, or leave empty.





Word Frequency

Measure of word frequency in each corpus chunk.

Format as whitespace seperated words. Choose either a file upload or use the text box.

Relative Word Frequency

Measure of relative word frequency in each corpus chunk: frequency within a corpus chunk to global word frequency in the entire corpus.

Format as whitespace seperated words. Choose either a file upload or use the text box.
